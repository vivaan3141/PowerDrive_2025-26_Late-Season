## Team 13777X – Push Back (2025–2026) Season

This repository contains the program for **VEX Robotics Team 13777X** during the **2025–2026 Push Back** season. 

## Competition Code
- **Autonomous:** Pre-programmed paths during autonomous period
- **Driver Control:** Controls for driver during a match
- **Sensors:** Use of inertial and distance sensors.
- **Late-Season Changes:** Code updates to optimize and organize program.

## Code Structure
- Written in C++, which is great for embedded systems (hardware integration).
- `/src` –  Robot logic + control systems  
- `/json` – Connection with VEX library
- `/include` – Constants, configurations, and coordination between src files
